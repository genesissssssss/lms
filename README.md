# Django Learning Management System (LMS)

A fully functional Learning Management System with dark-themed UI, student/admin dashboards, course management, and user authentication.

## Features

- ✅ User authentication with role-based access (Student/Instructor/Admin)
- ✅ Student dashboard with course progress tracking
- ✅ Admin dashboard with platform management
- ✅ Course management (CRUD operations)
- ✅ Material and video uploads
- ✅ User management and role assignment
- ✅ Student analytics and progress tracking
- ✅ Achievement system for students
- ✅ Dark/Light theme toggle
- ✅ Responsive design with Tailwind CSS

## Tech Stack

- Django 4.2
- Tailwind CSS
- PostgreSQL (Production)
- SQLite (Development)
- WhiteNoise for static files
- Vercel for deployment
